Fantasy Islands

Island Exploration Game with Obstacles

An exploration game created with Unreal 5 game engine. Gameplay includes
exploring floating islands and getting past obstacles such as moving flatforms,
shooting barrels, rolling boulders, and scaling castle wall.

Making this game is part of my learning journey to develop games with the Unreal engine :).
There are common video game issues (as seen in the playthough video) such as the camera colliding with game objects,
the character can unrealisticcally push huge boulders rolling downhill, etc. I'm just getting started
developing with Unreal and plan to make more fun projects in the near future.
